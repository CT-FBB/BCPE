## 1. คำนิยามและบทบาทอุปกรณ์ (Device Roles & Definitions)

ในการติดตั้งตามบ้านของลูกค้า บัญชีอินเทอร์เน็ตแต่ละราย (ระบุผ่าน **UserID / LOID**) อาจมีอุปกรณ์ออนไลน์พร้อมกันมากกว่า 1 ตัว โดยแยกบทบาทหน้าที่ดังนี้:

### 🔹 HGW (Home Gateway Unit - อุปกรณ์ตัวหลัก)
*   ทำหน้าที่เป็นเกตเวย์หลักในการเชื่อมต่อสัญญาณไฟเบอร์หรืออินเทอร์เน็ต PPPoE เข้ามาในบ้าน
*   **รุ่นอุปกรณ์หลัก:** 
    *   ตระกูล **`T`** ของ Humax (เช่น `T626ProV2`, `T623L`, `T626Pro`, `T625LM`) ซึ่งเป็น Routing ONTs
    *   Routing ONTs แบรนด์อื่น ๆ เช่น `GN542VF` (Skyworth), `F6600P` / `F688` (ZTE), `HG8247W5-6T` (Huawei)
    *   Legacy Bridge ONTs ตัวรับไฟเบอร์แบบสะพานของเดิม เช่น `HG8247H`, `F612C`, `HG8122`, `H662GR`, `GN2000*`, `H660*`, `HT803*`, `AN5506*`, `F668`, `HG624*`, `AN5121*`

### 🔹 Mesh (ตัวขยายสัญญาณแบบ Mesh)
*   ทำหน้าที่เป็นตัวขยายสัญญาณ Wi-Fi โดยแชร์ SSID เดียวกันและทำงานร่วมกับ Routing ONT หลักเป็นโครงข่ายอัจฉริยะ (Smart Mesh)
*   **รุ่นอุปกรณ์ Mesh:** 
    *   ตระกูล **`A`** ของ Humax (เช่น `A625M`, `A623`, `A662`, `A626M`) ซึ่งทำหน้าที่เป็นตัวลูกเสริม
    *   Mesh AP เฉพาะทางของแบรนด์อื่น เช่น `RN510` (Skyworth), `ZXHN H3601P V9` (ZTE)
    *   ONT ตัวที่สองที่เป็นรุ่นเดียวกันในโหมด `as_mesh` (เช่น ตัวที่สองของ `T626ProV2` หรือ `T626Pro`)

### 🔹 AP (Access Point ในชุด 2-Box)
*   ทำหน้าที่เป็นตัวปล่อยสัญญาณไร้สาย (ตัวที่ 2) พ่วงหลัง Legacy Bridge ONT ซึ่งระบบ provisioning/ACS จะแยกเป็นหมวดหมู่ AP ชัดเจน ไม่เรียกว่า Mesh
*   **รุ่นอุปกรณ์ AP หลักและแบรนด์ที่ถูกต้อง:**
    *   `WR6221-T` : แบรนด์ **Turbo-speed** (AP กล่องที่สองที่พบบ่อยที่สุด)
    *   `RE1200R4GC-V3` : แบรนด์ **Humax** (WiFi Extender)
    *   `T3ATv2` : แบรนด์ **Humax** (AP)
    *   `ST-R4D` : แบรนด์ **Sercomm** (AP)
    *   `HG180Ev2` : แบรนด์ **Fiberhome** (Access Point)

## 2. รูปแบบการติดตั้งจับคู่อุปกรณ์ (Pairing Types)

เมื่อดึงรายการ LOID ที่มีอุปกรณ์ออนไลน์ $\ge 2$ เครื่อง (รวมทั้งหมด 356,308 ราย) ระบบจะกรองและจัดกลุ่มประเภทชุดติดตั้งเป็น 4 รูปแบบหลัก:

1.  **`routing_ont_mesh` (Routing ONT + Mesh AP):** 
    *   *กฎการกรอง:* ONT ตัวหลักทำหน้าที่เกตเวย์ (รุ่นตระกูล `T` หรือ `GN542` / `F6600`) คู่กับตัวขยายสัญญาณ Mesh (รุ่นตระกูล `A` หรือ `RN510` / `H3601P`)
    *   *บทบาท:* ONT = HGW, ตัวขยาย = Mesh
2.  **`2-box` (Legacy ONT + AP):**
    *   *กฎการกรอง:* โมเด็มไฟเบอร์บริดจ์รุ่นเก่า (เช่น `HG8247H`, `F612C`, `HG8122`) คู่กับ AP (เช่น `WR6221-T`, `RE1200R4GC-V3`, `T3ATv2`, `ST-R4D`)
    *   *บทบาท:* Legacy ONT = HGW, ตัวขยาย = AP
3.  **`as_mesh` (ONT-ONT เดียวกันทำ Mesh):**
    *   *กฎการกรอง:* อุปกรณ์รุ่นเดียวกันออนไลน์ใน LOID เดียวกัน (เช่น `T626ProV2` ➔ `T626ProV2` หรือ `T626Pro` ➔ `T626Pro`)
    *   *บทบาท:* ตัวหลัก = HGW, ตัวพ่วงเสริม = Mesh
4.  **`other_pairing` (เคสข้อยกเว้นพิเศษ):**
    *   *กฎการกรอง:* เคสจับคู่ที่ไม่ได้อยู่ภายใต้กฎ 3 ข้อหลัก (เช่น พ่วงอุปกรณ์เก่าเฉพาะรุ่นที่ไม่มีข้อมูลโมเดลหรือเป็นเครื่องทดสอบในคลัง)
